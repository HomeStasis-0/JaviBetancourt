Portfolio

My current research revolves around using the immersive component of video games, to offer a more engaging and effective method for stress management, that traditional stress and anxiety interventions lack.

CyberHam is a project that I worked on with other members of the Texas A&M Cybersecurity Club's Tech Committee. We were tasked with converting our SQL database into self-managed Elasticsearch instances using Python, Elasticsearch Queries, and AWS. The project would allow the club officers to see club data a lot more effectively while also allowing us to add more questions to better gage the needs of the clubs as a whole.

Wordle was the final project for my Programming Languages course. We were tasked with designing the user interface and then code the entirety of the game logic, using Java and JavaFX. The project allowed for an entry level gateway into game development and into making a user friendly interface.

Flappy Bird was a personal project that I took on in order to learn more of C# and Unity. The project was a great way to learn about game development and the importance of making a fun and engaging game. Thanks to this project I was able to learn more about game physics which aided me in other projects such as Wordle.
